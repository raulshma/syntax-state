## Color Models

CSS provides three main ways to define colors: **RGB**, **HSL**, and **Hex**. Each has its own strengths, and you can use whichever feels most natural to you!

### RGB - Red, Green, Blue

RGB works like mixing colored lights. Imagine three flashlights (red, green, and blue) - by adjusting their brightness from 0 to 255, you can create any color!

```css
.red-text {
  color: rgb(255, 0, 0);
}

.purple-text {
  color: rgb(128, 0, 128);
}

.white-text {
  color: rgb(255, 255, 255);
}
```

<KeyConcept title="RGB Values">
- **First number**: Red (0-255)
- **Second number**: Green (0-255)
- **Third number**: Blue (0-255)
- **rgb(0, 0, 0)**: Black (no light)
- **rgb(255, 255, 255)**: White (all lights at full)
</KeyConcept>

### HSL - Hue, Saturation, Lightness

HSL is like describing a color the way humans naturally think about it: "What color is it? How vibrant? How bright?"

```css
.blue-text {
  color: hsl(240, 100%, 50%);
}

.light-blue {
  color: hsl(240, 100%, 75%);
}

.muted-blue {
  color: hsl(240, 50%, 50%);
}
```

<KeyConcept title="HSL Values">
- **Hue**: The color itself (0-360 degrees on a color wheel)
  - 0Â° = Red, 120Â° = Green, 240Â° = Blue
- **Saturation**: How vibrant (0% = gray, 100% = full color)
- **Lightness**: How bright (0% = black, 50% = normal, 100% = white)
</KeyConcept>

<InfoBox type="tip" title="When to Use HSL">
HSL is great when you want to create color variations! Want a lighter version? Just increase the lightness. Want a more muted version? Decrease the saturation.
</InfoBox>

### Hex - Hexadecimal

Hex codes are the most common way to write colors in CSS. They're compact and widely used in design tools.

```css
.red-text {
  color: #FF0000;
}

.green-text {
  color: #00FF00;
}

.blue-text {
  color: #0000FF;
}
```

<KeyConcept title="Understanding Hex Codes">
A hex code has 6 characters: **#RRGGBB**
- First 2 characters: Red (00-FF)
- Middle 2 characters: Green (00-FF)
- Last 2 characters: Blue (00-FF)

**Shorthand**: If pairs match, you can shorten it!

</KeyConcept>

## Opacity and Transparency

Sometimes you want colors to be see-through, like looking through colored glass. CSS provides two ways to add transparency!

### RGBA and HSLA

Add an **alpha channel** (transparency) to RGB or HSL by using `rgba()` or `hsla()`.

```css
.semi-transparent-red {
  background-color: rgba(255, 0, 0, 0.5);
}

.semi-transparent-blue {
  background-color: hsla(240, 100%, 50%, 0.5);
}
```

<KeyConcept title="Alpha Values">
The alpha value controls transparency:
- **1** or **100%**: Fully opaque (solid)
- **0.5** or **50%**: Half transparent
- **0** or **0%**: Fully transparent (invisible)
</KeyConcept>

### The opacity Property

The `opacity` property makes an **entire element** transparent, including all its children.

```css
.faded-box {
  opacity: 0.7;
}

.very-faded {
  opacity: 0.3;
}
```

<InfoBox type="warning" title="RGBA vs Opacity">
**RGBA/HSLA**: Only makes the color transparent
**opacity**: Makes the entire element and all its children transparent

Use RGBA/HSLA when you only want the background transparent but text solid!
</InfoBox>
