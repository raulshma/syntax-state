# CSS Typography

Welcome to CSS Typography! Think of typography as the **art of making text beautiful and readable**. Just like choosing the right font for a poster or a book cover, CSS gives you the power to control how text looks on your web pages.

In this lesson, you'll learn how to change fonts, adjust sizes, align text, and add decorative effects. By the end, you'll be able to make your text look professional and engaging!

<InfoBox type="tip" title="Why Learn Typography?">
Typography is one of the most important aspects of web design. Good typography makes your content easy to read and helps communicate your message effectively. Poor typography can make even great content hard to consume.
</InfoBox>

## Font Properties

Font properties control the **typeface** and **appearance** of your text. Let's explore the most important ones!

### font-family

The **font-family** property lets you choose which typeface to use. Think of it like choosing between handwriting, print, or fancy calligraphy.

```css
p {
  font-family: Arial, sans-serif;
}
```

<KeyConcept title="Font Stacks">
Always provide multiple fonts as a **fallback stack**. If the first font isn't available on the user's computer, the browser will try the next one, and so on.
</KeyConcept>

Common font families:
- **Serif fonts**: Times New Roman, Georgia (have little "feet" on letters)
- **Sans-serif fonts**: Arial, Helvetica (clean, no "feet")
- **Monospace fonts**: Courier, Consolas (all letters same width, like typewriters)

### font-size

The **font-size** property controls how big or small your text appears.

```css
h1 {
  font-size: 32px;
}

p {
  font-size: 16px;
}
```

Common units:
- **px** (pixels): Fixed size, like `16px`
- **em**: Relative to parent font size, like `1.5em` (1.5 times parent size)
- **rem**: Relative to root font size, like `1rem` (usually 16px)

<InfoBox type="tip" title="Which Unit Should I Use?">
For beginners, **px** is easiest to understand. As you advance, **rem** is great for responsive designs because it scales with user preferences.
</InfoBox>

### font-weight

The **font-weight** property controls how **bold** or **light** your text appears.

```css
.bold-text {
  font-weight: bold;
}

.light-text {
  font-weight: 300;
}
```

Common values:
- **normal** or **400**: Regular weight
- **bold** or **700**: Bold text
- **100-900**: Numeric values (if the font supports them)

### font-style

The **font-style** property makes text **italic** or **oblique** (slanted).

```css
.italic {
  font-style: italic;
}

.normal {
  font-style: normal;
}
```

## Text Properties

Text properties control how text is **aligned**, **decorated**, and **transformed**.

### text-align

The **text-align** property controls the horizontal alignment of text within its container.

```css
.center {
  text-align: center;
}

.right {
  text-align: right;
}

.left {
  text-align: left;
}
```

Think of it like the alignment buttons in a word processor!

### text-decoration

The **text-decoration** property adds lines to your text.

```css
.underline {
  text-decoration: underline;
}

.line-through {
  text-decoration: line-through;
}

.no-decoration {
  text-decoration: none;
}
```

Common uses:
- **underline**: For emphasis or links
- **line-through**: For showing deleted or discounted text
- **none**: To remove default link underlines

<InfoBox type="warning" title="Be Careful with Underlines">
Users expect underlined text to be clickable links. Only underline text if it's actually a link, or you might confuse your visitors!
</InfoBox>

### text-transform

The **text-transform** property changes the capitalization of text.

```css
.uppercase {
  text-transform: uppercase;
}

.lowercase {
  text-transform: lowercase;
}

.capitalize {
  text-transform: capitalize;
}
```

Examples:
- **uppercase**: "hello world" → "HELLO WORLD"
- **lowercase**: "HELLO WORLD" → "hello world"
- **capitalize**: "hello world" → "Hello World"

## Combining Font and Text Properties

Now let's put everything together! You can combine multiple properties to create beautiful text styles.

```css
.heading {
  font-family: Georgia, serif;
  font-size: 36px;
  font-weight: bold;
  text-align: center;
  text-transform: uppercase;
}

.subtitle {
  font-family: Arial, sans-serif;
  font-size: 18px;
  font-weight: 300;
  font-style: italic;
  text-align: center;
}
```

<KeyConcept title="Typography Hierarchy">
Use different font sizes, weights, and styles to create a **visual hierarchy**. This helps readers understand which content is most important:
- **Headings**: Larger, bolder
- **Body text**: Medium size, regular weight
- **Captions**: Smaller, lighter weight
</KeyConcept>

## Key Takeaways

<KeyConcept title="Summary">
**Font Properties:**
- `font-family`: Choose the typeface (with fallback fonts)
- `font-size`: Control text size (px, em, rem)
- `font-weight`: Make text bold or light (normal, bold, 100-900)
- `font-style`: Make text italic or normal

**Text Properties:**
- `text-align`: Align text (left, center, right)
- `text-decoration`: Add lines (underline, line-through, none)
- `text-transform`: Change capitalization (uppercase, lowercase, capitalize)

**Best Practices:**
- Always provide fallback fonts in font-family
- Use appropriate font sizes for readability (16px minimum for body text)
- Create visual hierarchy with different sizes and weights
- Be careful with underlines (users expect them to be links)
</KeyConcept>
