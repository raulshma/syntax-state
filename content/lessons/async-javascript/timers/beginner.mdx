JavaScript timers work exactly like this!

## setTimeout: Do Something Later

`setTimeout` runs code **once** after a delay:

```javascript
// Show a message after 3 seconds
setTimeout(function () {
  console.log("Hello! 3 seconds have passed!");
}, 3000); // 3000 milliseconds = 3 seconds

console.log("Timer started...");
// Output: "Timer started..." immediately
// Then after 3 seconds: "Hello! 3 seconds have passed!"
```

## setInterval: Do Something Repeatedly

`setInterval` runs code **over and over** at regular intervals:

```javascript
// Count every second
let count = 0;

setInterval(function () {
  count++;
  console.log("Count:", count);
}, 1000); // Every 1000ms = 1 second

// Output: "Count: 1", "Count: 2", "Count: 3"... forever!
```

## Stopping a Timer

Every timer has an ID. Use it to stop the timer:

```javascript
// For setTimeout
const timerId = setTimeout(() => {
  console.log("This will run");
}, 1000);

clearTimeout(timerId); // Cancel it! Nothing happens.

// For setInterval
const intervalId = setInterval(() => {
  console.log("Tick!");
}, 1000);

// Stop after 5 seconds
setTimeout(() => {
  clearInterval(intervalId);
  console.log("Timer stopped!");
}, 5000);
```
