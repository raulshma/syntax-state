## What is a Blazor Component?

A component is a **self-contained chunk of UI** written in C# and HTML. Think of it like a custom HTML tag that you create!

### Your First Component

```razor
<!-- Counter.razor -->
@page "/counter"

<h1>Counter</h1>
<p>Count: @currentCount</p>
<button @onclick="Increment">Click me!</button>

@code {
    private int currentCount = 0;

    void Increment()
    {
        currentCount++;
    }
}
```

**What's happening here?**

- `@page "/counter"` = This component has its own URL
- `@currentCount` = Display the variable in HTML
- `@onclick` = When clicked, run C# code!
- `@code { }` = Your C# logic lives here

## Component Parameters: Passing Data In

Think of parameters like **function arguments** but for components!

### Creating a Greeting Component

```razor
<!-- Greeting.razor -->
<div class="greeting">
    <h2>Hello, @Name!</h2>
    <p>@Message</p>
</div>

@code {
    [Parameter]
    public string Name { get; set; } = "Friend";

    [Parameter]
    public string Message { get; set; } = "Welcome!";
}
```

### Using Your Component

```razor
<!-- Parent.razor -->
<Greeting Name="Alice" Message="Have a great day!" />
<Greeting Name="Bob" Message="Thanks for visiting!" />
<Greeting Name="Charlie" />  <!-- Uses default message -->
```

**Result:**

```
Hello, Alice!
Have a great day!

Hello, Bob!
Thanks for visiting!

Hello, Charlie!
Welcome!
```

## Data Binding: The Two-Way Street üîÑ

**Binding** connects your HTML to your C# variables automatically!

### One-Way Binding (Read-Only)

```razor
<p>Your name is: @userName</p>

@code {
    private string userName = "Alice";
}
```

### Two-Way Binding (Read & Write)

```razor
<input @bind="userName" />
<p>Hello, @userName!</p>

@code {
    private string userName = "Alice";
}
```

**Try it!** When you type in the input, the paragraph updates automatically! ‚ú®

## Binding to Different Events

By default, binding updates when you **leave the input** (blur). But you can change this!

```razor
<!-- Updates on every keystroke! -->
<input @bind="searchText" @bind:event="oninput" />
<p>Searching for: @searchText</p>

@code {
    private string searchText = "";
}
```

## Binding Different Data Types

Blazor automatically converts between HTML and C#:

```razor
<!-- String -->
<input @bind="name" type="text" />

<!-- Number -->
<input @bind="age" type="number" />

<!-- Date -->
<input @bind="birthday" type="date" />

<!-- Checkbox -->
<input @bind="isStudent" type="checkbox" />

@code {
    private string name = "";
    private int age = 0;
    private DateTime birthday = DateTime.Today;
    private bool isStudent = false;
}
```

## Real-World Example: A Simple Form

Let's build a contact form that uses everything we've learned!

```razor
<!-- ContactForm.razor -->
<div class="form-container">
    <h2>Contact Us</h2>

    <label>Name:</label>
    <input @bind="contactName" />

    <label>Email:</label>
    <input @bind="email" type="email" />

    <label>Message:</label>
    <textarea @bind="message" rows="4"></textarea>

    <button @onclick="SubmitForm">Submit</button>

    @if (submitted)
    {
        <div class="success">
            ‚úÖ Thanks, @contactName! We'll contact you at @email
        </div>
    }
</div>

@code {
    private string contactName = "";
    private string email = "";
    private string message = "";
    private bool submitted = false;

    void SubmitForm()
    {
        submitted = true;
    }
}
```

**What's Happening:**

1. Each input is bound to a variable
2. When you type, variables update
3. Click submit ‚Üí `submitted` becomes `true`
4. Conditional rendering shows success message!

## Component Reusability

The beauty of components is **reusing** them! Here's a reusable button:

```razor
<!-- FancyButton.razor -->
<button class="fancy-btn @ColorClass" @onclick="HandleClick">
    @Text
</button>

@code {
    [Parameter]
    public string Text { get; set; } = "Click";

    [Parameter]
    public string ColorClass { get; set; } = "blue";

    [Parameter]
    public EventCallback OnClick { get; set; }

    async Task HandleClick()
    {
        await OnClick.InvokeAsync();
    }
}
```

**Use it anywhere:**

```razor
<FancyButton Text="Save" ColorClass="green" OnClick="SaveData" />
<FancyButton Text="Delete" ColorClass="red" OnClick="DeleteData" />
<FancyButton Text="Cancel" ColorClass="gray" OnClick="Cancel" />
```
