# Basic Numeric Functions

Master essential math functions for data manipulation!

##  Rounding Functions

### ROUND - Round to Specific Decimals

```sql
SELECT ROUND(3.14159, 2);  -- 3.14
SELECT ROUND(3.145, 2);    -- 3.15 (rounds up)
SELECT ROUND(123.456, 0);  -- 123
SELECT ROUND(123.456, -1); -- 120 (round to tens)
```

### CEILING - Round Up

```sql
SELECT CEILING(4.1);   -- 5
SELECT CEILING(4.9);   -- 5
SELECT CEILING(-4.1);  -- -4
```

### FLOOR - Round Down

```sql
SELECT FLOOR(4.9);   -- 4
SELECT FLOOR(4.1);   -- 4
SELECT FLOOR(-4.1);  -- -5
```

##  Quick Comparison

| Value | ROUND | CEILING | FLOOR |
| ----- | ----- | ------- | ----- |
| 4.2   | 4     | 5       | 4     |
| 4.8   | 5     | 5       | 4     |
| -4.2  | -4    | -4      | -5    |

##  ABS - Absolute Value

Remove the negative sign:

```sql
SELECT ABS(-5);    -- 5
SELECT ABS(5);     -- 5
SELECT ABS(-3.14); -- 3.14
```

##  Practical Examples

```sql
-- Round prices to 2 decimal places
SELECT name, ROUND(price, 2) AS formatted_price FROM products;

-- Calculate shipping (round up to nearest whole)
SELECT CEILING(weight / 10.0) AS shipping_units FROM packages;

-- Find distance from a value
SELECT ABS(target - actual) AS difference FROM measurements;
```

##  Basic Math

```sql
SELECT 10 + 5;   -- 15 (addition)
SELECT 10 - 5;   -- 5  (subtraction)
SELECT 10 * 5;   -- 50 (multiplication)
SELECT 10 / 3;   -- 3  (integer division in some DBs)
SELECT 10.0 / 3; -- 3.333... (decimal division)
SELECT 10 % 3;   -- 1  (modulo/remainder)
```
