# DATEPART and DATEADD: Date Manipulation

Extract parts of dates and perform date math!

##  DATEPART: Extract Components

```sql
SELECT DATEPART(YEAR, '2024-03-15');   -- 2024
SELECT DATEPART(MONTH, '2024-03-15');  -- 3
SELECT DATEPART(DAY, '2024-03-15');    -- 15
SELECT DATEPART(HOUR, '2024-03-15 14:30:00');  -- 14
```

##  Common Date Parts

| Part    | Returns |
| ------- | ------- |
| YEAR    | 2024    |
| MONTH   | 1-12    |
| DAY     | 1-31    |
| HOUR    | 0-23    |
| MINUTE  | 0-59    |
| SECOND  | 0-59    |
| WEEKDAY | 1-7     |
| WEEK    | 1-53    |
| QUARTER | 1-4     |

##  DATEADD: Add Time

```sql
SELECT DATEADD(DAY, 7, '2024-03-15');    -- 2024-03-22
SELECT DATEADD(MONTH, 1, '2024-03-15');  -- 2024-04-15
SELECT DATEADD(YEAR, -1, '2024-03-15');  -- 2023-03-15
```

##  DATEDIFF: Calculate Difference

```sql
SELECT DATEDIFF(DAY, '2024-03-01', '2024-03-15');   -- 14
SELECT DATEDIFF(MONTH, '2024-01-15', '2024-06-20'); -- 5
SELECT DATEDIFF(YEAR, '2020-01-01', '2024-12-31'); -- 4
```

##  Practical Examples

```sql
-- Orders in last 7 days
SELECT * FROM orders
WHERE order_date >= DATEADD(DAY, -7, GETDATE());

-- Group by month
SELECT
    DATEPART(YEAR, order_date) AS year,
    DATEPART(MONTH, order_date) AS month,
    COUNT(*) AS order_count
FROM orders
GROUP BY DATEPART(YEAR, order_date), DATEPART(MONTH, order_date);

-- Calculate age
SELECT DATEDIFF(YEAR, birth_date, GETDATE()) AS age FROM users;
```

##  Database Variations

```sql
-- PostgreSQL uses EXTRACT
SELECT EXTRACT(YEAR FROM date_column);
SELECT EXTRACT(MONTH FROM date_column);

-- MySQL uses both
SELECT YEAR(date_column);
SELECT MONTH(date_column);
```
