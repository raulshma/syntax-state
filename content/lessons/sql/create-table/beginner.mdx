# CREATE TABLE: Building Your Data Structure

Learn to create tables - the foundation of any database!

## Ô∏è What is CREATE TABLE?

CREATE TABLE makes a new table to store your data. Think of it as creating a new spreadsheet with specific column headers.

##  Basic Syntax

```sql
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype
);
```

##  Your First Table

```sql
CREATE TABLE customers (
    id INT,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    email VARCHAR(100),
    created_at DATE
);
```

This creates a table with 5 columns!

##  Choosing Data Types

| Data            | Type to Use   | Example       |
| --------------- | ------------- | ------------- |
| Whole numbers   | INT           | Age, quantity |
| Decimal numbers | DECIMAL(10,2) | Price, salary |
| Short text      | VARCHAR(50)   | Name, city    |
| Long text       | TEXT          | Description   |
| Yes/No          | BOOLEAN       | is_active     |
| Dates           | DATE          | birth_date    |
| Date + Time     | DATETIME      | created_at    |

##  Primary Key

Every table needs a unique identifier:

```sql
CREATE TABLE products (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    price DECIMAL(10,2)
);
```

##  Auto-Incrementing IDs

Let the database assign IDs automatically:

```sql
-- SQL Server
CREATE TABLE orders (
    id INT IDENTITY(1,1) PRIMARY KEY,
    order_date DATE
);

-- PostgreSQL
CREATE TABLE orders (
    id SERIAL PRIMARY KEY,
    order_date DATE
);

-- MySQL
CREATE TABLE orders (
    id INT AUTO_INCREMENT PRIMARY KEY,
    order_date DATE
);
```

##  NOT NULL Constraint

Require values in important columns:

```sql
CREATE TABLE users (
    id INT PRIMARY KEY,
    email VARCHAR(100) NOT NULL,  -- Must have a value
    nickname VARCHAR(50)          -- Optional
);
```

##  Complete Example

```sql
CREATE TABLE employees (
    id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    hire_date DATE,
    salary DECIMAL(10,2),
    is_active BOOLEAN DEFAULT TRUE
);
```
